NNWeb (Beta)

NNWeb is a webpage that allows users to build a Neural Network and see how it back propagates. It is a learning tool I've been working on for TDAC to illustrate the gradient issues that occur when using many layers. This is NOT the finished solution but I thought it might be interesting to some people.
